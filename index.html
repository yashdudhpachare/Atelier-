From a bare room photo to a finished, shoppable room — without hiring a designer.

Atelier is a validation MVP for AI interior design. The bet: most people can't visualise a room makeover, and won't pay a designer to find out. So I built the whole loop myself — photo in, redesign and a real shopping list out.

This page tells the build story — the product, the AI pipeline behind it, and the n8n engine that stitches it all together.

How it works

Four steps.
No designer required.

The user's job is kept tiny — a photo, a vibe, a budget. Everything hard happens behind the curtain. The entire interaction is designed to feel like texting a friend who happens to be an interior designer.

03
The AI pipeline

Three models,
one continuous brief.

Behind the four taps, three AI stages hand a structured brief down the line — each one reading the previous stage's JSON, never starting from scratch. Vision reads the room, styling translates the vibe into a spec, and render paints the photoreal scene.

Stage 01 · VisionGPT-4o

Read the room

Input

Photo of an empty bedroom — white walls, wood floor, single window.

Output JSON
{ "walls":"white",
  "floor":"oak",
  "light":"natural-east",
  "dimensions":"3.6 × 4.2 m" }
Stage 02 · StylingGPT-4o

Translate vibe → spec

Input

User vibe + room JSON + budget — “cozy, warm, scandinavian” / €1,200.

Output JSON
{ "palette":["oat","olive","black"],
  "items":["low platform bed",
    "wool rug","floor lamp"] }
Stage 03 · RenderImagen 3

Photoreal scene

Input

Spec JSON + style brief + a photo-style render prompt.

Output
1024 × 1024 PNG
real-product, real-light render
→ handed to product matching
04
The orchestration

n8n is the engine room.

No bespoke backend. A single webhook kicks off an n8n workflow that fans out across the AI services, matches real products, and hands a finished response back to the app. Each node does one job and passes clean JSON to the next.

Atelier proves the experience is buildable by one person with today's AI plumbing. From here, three threads are worth pulling.
